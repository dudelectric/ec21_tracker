John Main
Hi Joe, Unfortunately this type of module is a single cell charger only. This is because it can only be set to one charge voltage detection level 4.2V and can only monitor one cell voltage.

You need a different chip that performs charge balancing used in drones for the higher voltage battery packs. They basically monitor each cell individually for chareg state and stop each one overcharging. Also if one has a lower voltage than the others all are discharged to that voltage.
